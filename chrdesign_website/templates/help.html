Chromosome Designer is a tool to design synthetic chromosomes intended to be constructed in yeast via in vivo assembly (Fig. 1). This tool provides the primers required to create all the DNA fragments needed for the assembly, as well as the chromosome map that should result after assembly.

1 Input

For each of the fragments, Chromosome Designer requires the following information:
  • Fragment name
  • Genbank file of the DNA that would serve as template to amplify the fragment
  • Start position of the fragment within this template
  • End position of the fragment within this template
Example
Imagine that we want to create a simple plasmid containing a yeast selection marker (URA3) and a yeast replication origin (2micron). Based on the maps shown in Fig. 2, the input information will be:

Fragment 1 Fragment 2
Fragment name URA3 2micron
Start position 14 1056
End position 1072 2038
Genbank file pYTK074.gb pUDE1217.gb



1.1 How to enter input?
There are two ways to enter information to Chromosome Designer:
  1. Using a form
  2. Using an excel file

A) Enter fragments through form
In Fig. 3, you can see how to enter information through a form using the "Enter fragments" tab.



B) Enter fragments from file
To enter information using a file, you can download a template excel file in the "Enter fragments from file" tab (Fig. 4). After filling in the fragments' information (Fig. 5), you can upload the file and the corresponding maps using the "Enter fragments from file" tab (Fig. 4). Make sure you provide all the maps specified in your excel file with the correct spelling.



1.2 How to modify primer settings?
Chromosome Designer allows to modify several primer settings:
  • Minimum GC content. Default: 40%.
  • Maximum GC content. Default: 60%.
  • Minimum annealing temperature of the primer pair. Default: 58째C.
  • Maximum annealing temperature of the primer pair. Default: 64째C.
  • Maximum melting temperature difference between the two primers in the pair. Default: 5째C.
If primers meeting the specified requirments cannot be found, Chromosome Designer searches for primers in a range of 100 nucleotides upstream and downstream the start and end of the fragment, respectively (this option is activated by default).

Changing primer settings can be done for all the fragments at once, or for individual fragments.

A) Changing primer settings for all fragments
When entering fragments from a form, general primer settings can be modified under the "Primer settings" tab (Fig. 6).


When entering fragments from a file, general primer settings can be modified under the "General primer settings" tab of the excel file (Fig. 7).



B) Changing primer settings for individual fragments
When entering fragments from a form, primer settings for individual fragments can be modified by expanding the "Adjust primers settings for this fragment" collapsible of the fragment of interest (Fig. 8). Make sure that the box to apply changes it ticked.


When entering fragments from a file, primer settings of individual fragments can be modified by adding the desired values for the fragment of interest in the "Fragments" tab of the excel file (Fig. 9). If no value is specified for a certain setting, the value from the "General primer settings" tab will be used. In the example below, the minimum and maximum GC content of the URA3 primers have been set to 30% and 70%, respectively. The maximum melting temperature difference between the two primers of the 2micron fragment has been reduced to 3째C.




2. Output

Chromosome Designer generates three output files:
  • A Primer Database
  • A Synthetic Chromosome Map
  • An Input Data Summary

2.1 Primer Database
This is an excel file containing the primers required to generate all the fragments that will be used for assembly. For each primer the following information is given:
  • Primer name
  • Primer sequence (including the homologous recombination overhangs used for assembly, in capital letters)
  • Primer length
  • Annealing temperature of the primer pair
  • Target region to be amplified
If primers cannot be found for one or more fragments, try again releasing some primer constraints (modify primer settings).

2.2 Synthetic Chromosome Map
This is a genbank file of the synthetic chromosome that should result after assembly. If primers cannot be found for one or more fragments, the map is not generated.




2.3 Input Data Summary
This is an excel file containing the information entered for each of the fragments. This file can be modified and used as input in the "Enter fragments from file" tab.
